# Locket Widget 2.0

A polished, responsive UI showcase inspired by the Locket experience. This project highlights a modern dashboard layout with a social feed, friends view, and widget gallery built with React and Vite.

## Live Demo

https://anshu706.github.io/LOCKETWIDGET2.0

## Features

- Elegant sidebar navigation and dashboard layout
- Feed, friends, and widgets views with rich UI elements
- Interactive widget modal details
- Responsive grid layouts and smooth animations
- Custom theme tokens and typography

## Tech Stack

- React 18
- Vite 5
- JavaScript (ES Modules)
- gh-pages (deployment)

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```

## Deployment

This project is configured for GitHub Pages deployment.

```bash
npm run deploy
```
