## ✨ Features

- 📸 **Live Camera Integration:** Access your webcam directly from the browser and capture moments to share with your friends.
- 📰 **Dynamic Feed:** A fully interactive feed showcasing the latest photo updates from your circle.
- 👥 **Friends Management:** View and interact with your closest friends in a beautiful interface.
- 📱 **Widgets Showcase:** Preview how photos appear across various devices, including Home Screen, Lock Screen, and Dynamic Island.
- 👤 **Profile Statistics:** Keep track of your shared photos, connections, and total reactions.
- 🎨 **Stunning UI/UX:** Built with modern design principles featuring glassmorphism, fluid animations (`fadeUp`, `popIn`, `slideLeft`), and a sleek dark theme.

## 💻 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anshu706/LOCKETWIDGET2.0.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd LOCKETWIDGET2.0
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in your browser:**
   Open `http://localhost:5173` to view the application in your browser.

## 📦 Build for Production

To create a production-ready build, run:
```bash
npm run build
```
This will generate an optimized bundle in the `dist` directory.
